## 🚀 Currently Building

### CaptionShroud

An AI-powered platform that helps creators automatically generate, edit and export animated captions for videos.

**Core workflow:**

```text
Video Upload → Audio Extraction → AI Transcription
→ Caption Editor → Animated Video Export
```

**Built with:**

`React` • `Node.js` • `Express` • `MongoDB` • `FFmpeg` • `Gemini AI` • `ImageKit`
